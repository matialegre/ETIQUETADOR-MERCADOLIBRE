Instalación - Versión DEPÓSITO

Nota: Esta versión está configurada para uso en el depósito principal con base de datos y API locales.
Paso 1: Descargar Archivos
  1. Descargar DISTRIBUCION_DEPOSITO.zip
  2. Extraer en una carpeta (ej: C:\MundoOutdoor\)
  3. Verificar que contenga:
    • Cliente_Matias_GUI_v3_DEPOSITO.exe
    • INICIAR_DEPOSITO.bat
    • README_DEPOSITO.txt
Paso 2: Verificar Requisitos
Software Necesario:
  • Windows 10/11
  • SQL Server local
  • ODBC Driver 17 for SQL Server
  • Impresora ZPL (para etiquetas)
Conexiones Requeridas:
  • Base de datos DRAGONFISH_DEPOSITO
  • API Dragonfish local
  • Internet (para MercadoLibre)
  • Red local (para impresora)
Paso 3: Primera Ejecución
  1. Hacer doble clic en INICIAR_DEPOSITO.bat
  2. Verificar que aparezca la ventana del sistema
  3. Comprobar conexión a base de datos
  4. Probar carga de pedidos con el botón "Cargar"
¡Listo! El sistema DEPÓSITO está configurado y funcionando.

Instalación - Versión CABA

Importante: Esta versión requiere conexión a servidores remotos. Verificar conectividad de red.
Paso 1: Descargar Archivos
  1. Descargar DISTRIBUCION_CABA.zip
  2. Extraer en una carpeta (ej: C:\MundoOutdoor\)
  3. Verificar que contenga:
    • Cliente_Matias_GUI_v3_CABA.exe
    • INICIAR_CABA.bat
    • README_CABA.txt
Paso 2: Verificar Conectividad
Probar SQL Server:
Test-NetConnection 192.168.1.14 -Port 1433
Debe devolver: TcpTestSucceeded : True
Probar API Dragonfish:
ping 190.211.201.217
Debe responder sin pérdida de paquetes
Paso 3: Configuración de Red
Configuración Automática:

La versión CABA viene preconfigurada con:

  • SQL Server: 192.168.1.14,1433
  • Base de datos: DRAGONFISH_MUNDOCAB
  • Usuario SQL: cliente_app
  • API Dragonfish: http://190.211.201.217:8009
Paso 4: Primera Ejecución
  1. Hacer doble clic en INICIAR_CABA.bat
  2. Verificar conexión a SQL Server remoto
  3. Comprobar acceso a API Dragonfish
  4. Probar carga de pedidos CABA
¡Perfecto! El sistema CABA está conectado y funcionando.

Configuración de Impresora

Impresora ZPL Requerida

El sistema requiere una impresora compatible con lenguaje ZPL para etiquetas de envío.

Modelos Recomendados:
  • Zebra ZD420
  • Zebra ZD620
  • Zebra GK420d
  • Cualquier impresora ZPL compatible
Configuración
  1. Instalar drivers de la impresora
  2. Configurar como impresora predeterminada
  3. Verificar que acepta comandos ZPL
  4. Probar impresión desde el sistema
El sistema detecta automáticamente la impresora ZPL configurada.

Problemas Comunes de Instalación

Síntomas: "Login failed" o "Server not found"
Solución:
  1. Verificar que SQL Server esté ejecutándose
  2. Comprobar firewall (puerto 1433)
  3. Verificar credenciales de usuario SQL
  4. Probar conectividad con Test-NetConnection

Síntomas: "Connection timeout" o "API not responding"
Solución:
  1. Verificar conexión a internet
  2. Comprobar que la IP sea accesible
  3. Revisar configuración de proxy/firewall
  4. Contactar administrador de red

Síntomas: "No ZPL printer found"
Solución:
  1. Verificar que la impresora esté encendida
  2. Instalar drivers correctos
  3. Configurar como impresora predeterminada
  4. Verificar compatibilidad ZPL
